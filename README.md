# My Notes iOS App with Firebase Backend

My Notes is a simple and efficient note-taking iOS application developed using SwiftUI with Firebase as the backend service. The app is designed to help users easily create and manage their personal notes with a clean and minimal interface.

The application includes a secure authentication system where users can sign up and sign in using Firebase Authentication. After logging in, users can add new notes, update existing notes, or delete notes whenever they want. The app also allows users to archive notes so they can keep important notes stored without deleting them.

To improve usability, the app provides a search feature that helps users quickly find specific notes. Users can also manage their profile information, making the app more personalized. All notes and user data are stored and synchronized using Firebase Firestore, providing a reliable and scalable backend.

This project demonstrates the implementation of CRUD operations, Firebase integration, and SwiftUI-based user interface design, making it a practical example of building a modern iOS application with cloud backend services.
